# Simple Event Registration System (SERS)

A mini-project for Software Configuration Management course.

## Features
- User authentication (login)
- View list of upcoming events
- Register for any event (saved in registrations.json)

## Requirements (simple)
1. User must log in (username: admin, password: password123)
2. After login → view event dashboard
3. Click "Register" on any event → registration recorded

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Data stored in JSON files (simulated database)
